🤖 LLM Demo

Aplicación web desarrollada con Streamlit para demostrar conceptos fundamentales de los Large Language Models (LLM) y su funcionamiento.

📖 Descripción

Este proyecto tiene como objetivo presentar de forma sencilla e interactiva algunos conceptos esenciales relacionados con los LLM modernos, incluyendo:

¿Qué es un LLM?
Tokenización
Embeddings
Arquitectura Transformer
Predicción de tokens
Inferencia
Aplicaciones de los LLM

La aplicación fue desarrollada con fines educativos para apoyar cursos de Ciencia de Datos, Inteligencia Artificial y Analítica de Negocios.

🚀 Tecnologías utilizadas
Python
Streamlit
Pandas
NumPy
Scikit-Learn

⚙️ Instalación local

Clonar el repositorio:

git clone https://github.com/RichardMeza/LLM-Demo.git

Ingresar al directorio:

cd LLM-Demo

Instalar dependencias:

pip install -r requirements.txt

Ejecutar la aplicación:

streamlit run app.py
